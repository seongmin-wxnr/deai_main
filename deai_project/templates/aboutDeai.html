About

Deai는 무엇인가요?

Deai(데아이)는 일본어로 "만남"을 의미해요.
혼자 플레이하기 지루하거나, 실력에 맞는 파티원을 찾고 싶을 때 Deai에서 모집 글을 올리거나 원하는 파티에 참여해보세요.

티어 조건, 포지션, 게임 스타일까지 맞는 사람과 함께라면 게임이 더 재밌어집니다.

Features

주요 기능

📋
파티 모집
게임, 티어 조건, 모집 인원을 설정해 원하는 파티원을 모집하세요.
🙋
파티 참여
마음에 드는 모집글에 참여 신청해 바로 파티에 합류하세요.
💬
실시간 채팅
파티원들과 모집글 내 채팅방에서 실시간으로 소통하세요.
👥
친구 시스템
자주 함께하는 유저를 친구로 추가하고 요청을 주고받으세요.
🏅
게임 프로필
게임별 티어, 인게임 ID, 포지션을 등록해 나를 알려보세요.
🔔
알림
친구 요청이 오면 알림으로 바로 확인하고 수락하세요.

How to use

어떻게 시작하나요?

1
회원가입 & 게임 프로필 등록
닉네임과 이메일로 가입 후, 즐기는 게임의 티어와 포지션을 등록하세요.
2
모집글 탐색
게임별 탭에서 원하는 조건의 파티 모집글을 확인하세요.
3
파티 참여 또는 모집
마음에 드는 글에 참여하거나, 직접 파티원을 모집하는 글을 올리세요.
4
채팅으로 소통
모집글 채팅방에서 파티원들과 게임 전략을 논의하세요.
